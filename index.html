Mais Sobre

Nascido do Brasil em 2016, Vítor Tavares ou também conhecido como GamerJooj se interessou por programação, sendo assim logo começou a estudar sobre programação em Game Maker Language, utilizavel no Game Maker Studio, depois a C.Sharp, utilizavel na Unity, e agora programação HTML, CSS e Java Script.
